## NLP

### [sequence_labeling](./sequence_labeling)

**模型：**Bert + BiLSTM + CRF
**模型输入：**一段文本。
**模型输出：**文本包含的机构、人名、时间等实体。
    
    input text : 中共中央总书记、国家主席江泽民发表１９９８年新年讲话
    [
        {
            "end": 4,
            "words": "中共中央",

            "begin": 1
        },
        {
            "end": 15,
            "words": "江泽民",
            "type": "PER",
            "begin": 13
        },
        {
            "end": 22,

### [information_extraction](./information_extraction)

**模型：**Bert
**模型输入：**实体A，实体B，包含实体A和实体B的文本。
**模型输出：**文本包含的机构、人名、时间等实体。
